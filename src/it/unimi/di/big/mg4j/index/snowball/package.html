Snowball-based {@linkplain it.unimi.di.big.mg4j.index.TermProcessor term processors}.

Package Specification

This package gathers a number of algorithmic stemmer (most notably, the famous {@link it.unimi.di.big.mg4j.index.snowball.PorterStemmer}) automatically generated from scripts written in Snowball—a string language developped by Martin Porter to describe easily algorithmic stemmers. Please refer to the documentation on the Snowball site for information about the stemmers.
